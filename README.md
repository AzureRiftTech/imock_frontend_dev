# Frontend (Next.js)

This frontend is a Next.js App Router app using TypeScript, Tailwind CSS, shadcn/ui-style components (local), Radix UI primitives, and Framer Motion.

## Development

- Create an env file from [.env.example](.env.example):
	- `NEXT_PUBLIC_API_BASE_URL=http://localhost:3333`

Commands:

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
